The Key Elements Fall Concert Promotional Poster

I created this poster for The Key Elements Co-Ed A Cappella this past fall. It was for our annual Rocktoberfest concert, which was themed to Avatar! I additionally edited the background of each button, added overlays and effects using the apps Procreate and Repix, and created the poster's backdrop using Procreate.

The Key Elements Spring Concert Promotional Poster

This is a poster I created for The Key Elements Co-Ed A Cappella's 30th Anniversary Concert this past spring, BacKE to the Future! I additionally edited the background of each button that is featured in the grid and created the backdrop behind the grid. Since our theme was decades/Back To The Future/time travel, I tried to emulate themes from the movie, and I also picked backgrounds for each button in the grid based on the decade each member chose to be "from."

The Key Elements Spring Auditions Promotional Poster

I created this poster for my a cappella group's spring audition period. It features buttons from our fall concert, and the backdrop is a filtered button from the concert as well. The intent behind this design was to make it easy to interpret such that our group could maximize the number of auditions we had.
